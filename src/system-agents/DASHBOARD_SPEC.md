# AI Lead Strategies Master Dashboard Specification

## Overview

The Self-Healing System Dashboard is an **admin-only** control center that provides real-time visibility into all 5 platforms:

- **LeadSite.AI** (Tier 1)
- **LeadSite.IO** (Tier 2)
- **ClientContact.IO** (Tier 3)
- **VideoSite.IO** (Tier 4)
- **Tackle.AI** (Tier 5)

---

## Dashboard Access

**URL:** `https://admin.aileadstrategies.com/system` or `https://aileadstrategies.com/admin/system`

**Access Control:**
- Admin users only (role-based)
- Requires authentication
- IP whitelist recommended for production

## Real-Time Updates

The dashboard should use **WebSocket** or **Server-Sent Events (SSE)** for real-time updates:

```javascript
// WebSocket connection
const ws = new WebSocket('wss://api.aileadstrategies.com/system/live');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);

  switch (data.type) {
    case 'ALERT':
      updateAlertsPanel(data.alert);
      break;
    case 'METRIC':
      updateMetricsChart(data.metric);
      break;
    case 'REPAIR':
      updateRepairsPanel(data.repair);
      break;
    case 'HEALTH':
      updateHealthStatus(data.health);
      break;
  }
};
```
